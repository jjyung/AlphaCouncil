# 市場心理分析師 (Psychology Analyst) 開發方式

## 文件目的

本文件定義 AlphaCouncil 中市場心理分析師的角色定位、資料策略、指標設計與輸出格式。

市場心理分析師關注的是**市場在害怕什麼、偏好什麼、風險承受度如何變化**，以「可量化的數字型指標」為核心判讀工具，不處理法人流向、融資融券或借券賣出等籌碼資訊，也不處理新聞語氣或社群情緒（該部分由 News Analyst 負責）。

---

## 一、角色定位

市場心理分析師負責從波動指標、選擇權情緒、市場行為 proxy 與資金流向 proxy，判斷目前市場屬於恐慌、樂觀、觀望或風險轉折階段。

### 主要職責

- 分析市場波動是否擴張或收斂
- 分析風險偏好是 risk-on 還是 risk-off
- 分析選擇權市場反映的多空情緒
- 監控市場行為 proxy 指標的異常變化
- 產出 `psychology_report`

### 不負責事項

- 不分析誰在買、誰在賣（歸 Chip Analyst）
- 不處理三大法人、融資融券、借券賣出（歸 Chip Analyst）
- 不取代技術分析師的價格趨勢判斷（歸 Technical Analyst）
- 不解讀新聞語氣或社群情緒（歸 News Analyst）

## 三、資料來源策略

### 3.1 US 市場

- `yfinance`：取得 VIX (`^VIX`)、VVIX (`^VVIX`)、個股與 ETF options 資料
- VIX：作為恐慌程度與波動 regime 主指標
- VVIX（有條件使用）：代表 VIX 本身的波動率；僅在 VIX 快速拉升時用於觀察上升加速度，一般情況不納入判讀
- Put/Call Ratio（Volume / OI 雙口徑）：從 options chain 計算，作為風險偏好輔助訊號
- Options IV 指標：`atm_iv_near`（近月 ATM IV）、`atm_iv_next`（次月 ATM IV）、`iv_term_slope`（period structure 斜率）、`iv_rr_25d`（25-delta RR，反映方向偏好）、`iv_bf_25d`（25-delta BF，反映尾部厚薄）

### 3.2 TW 市場

台股有免費的官方心理指標可用，以 TAIFEX 提供的波動率指數與選擇權情緒資料為核心，搭配市場行為 proxy 補足。

#### 核心波動指標：TAIFEX 臺指選擇權波動率指數（台版 VIX）

- TAIFEX 提供「臺指選擇權波動率指數」，以 CBOE VIX 公式計算
- **資料端點**：月度 `.txt` 下載檔（Big5 編碼），格式為：
  - `https://www.taifex.com.tw/file/taifex/Dailydownload/vix/log2data/{YYYYMM}new.txt`
- 實務上需聚合最近多個月份檔案，建立至少 1 年歷史視窗供 percentile / z-score 使用
- 此為台股 Psychology Analyst 的**首要核心指標**
- 免費、官方、無需 API 金鑰

#### 核心選擇權情緒：TAIFEX Put/Call Ratio

- **端點**：`https://openapi.taifex.com.tw/v1/PutCallRatio`
- 提供臺指選擇權 Put/Call 比，含兩種口徑：
  - 成交量口徑：反映交易情緒
  - 未平倉量（OI）口徑：反映持有偏好
- API 回傳為百分比格式（如 `125.26` 代表 `125.26%`），實作時需除以 100 轉為比值
- 免費、官方、無需 API 金鑰

#### 輔助資金流 proxy：USD/TWD 匯率趨勢

- `yfinance`：`USDTWD=X` 取得匯率日線
- 匯率不算籌碼核心，定位為資金流向的間接觀測
- USD/TWD 持續升值（台幣貶值）可能暗示外資撤出、risk-off
- USD/TWD 持續貶值（台幣升值）可能暗示外資流入、risk-on

#### 市場行為 proxy（數字型）

以加權指數（`^TWII`）為基礎計算，從市場行為側面反映情緒狀態：

- **加權指數近 5 日實現波動率**：衡量短期波動是否擴張

此指標以 5 日（一週）為觀察視窗，定位為 **secondary proxy**，輔助核心波動與選擇權指標。

### 3.3 資料來源端點總整理

| 指標                     | 分類           | 資料來源       | 端點 / 方法                                            | 頻率 | 需要 API Key |
| ------------------------ | -------------- | -------------- | ------------------------------------------------------ | ---- | ------------ |
| 臺指選擇權波動率指數     | 核心           | TAIFEX         | `.txt` 月度下載檔（Big5）                               | 日   | 否           |
| 臺指選擇權 PCR（成交量） | 核心           | TAIFEX OpenAPI | `https://openapi.taifex.com.tw/v1/PutCallRatio`        | 日   | 否           |
| 臺指選擇權 PCR（OI）     | 核心           | TAIFEX OpenAPI | `https://openapi.taifex.com.tw/v1/PutCallRatio`        | 日   | 否           |
| USD/TWD 匯率             | 輔助           | yfinance       | `yf.Ticker("USDTWD=X")`                                | 日   | 否           |
| 加權指數 OHLCV           | 計算用         | yfinance       | `yf.Ticker("^TWII")`                                   | 日   | 否           |
| VIX (US)                 | 核心           | yfinance       | `yf.Ticker("^VIX")`                                    | 日   | 否           |
| VVIX (US)                | 輔助（有條件） | yfinance       | `yf.Ticker("^VVIX")`                                   | 日   | 否           |

### 3.4 選型原則

- 心理分析以**數字型指標**為核心，不依賴新聞語氣或社群情緒
- 台股已有官方免費 VIX 指標，不再需要純 proxy 替代
- 心理分析重點是 regime 判讀，不追求單一完美指標
- 台股 secondary proxy（實現波動率）需在報告中說明其為間接觀測
- 心理分析只回答風險偏好，不回答籌碼流向
- 核心指標（VIX、PCR）判讀優先於輔助指標（匯率、實現波動率），加權計分時核心指標權重為 2，輔助指標權重為 1

## 四、分析框架

### 4.1 核心問題

- 市場目前偏恐慌還是偏樂觀？
- 波動是否明顯放大？
- 選擇權市場偏避險還是偏投機？
- 市場行為 proxy 是否出現異常？
- 短期情緒是否與價格趨勢背離？

### 4.2 指標分組

#### TW 市場

| 分類           | 指標                             | 用途                         |
| -------------- | -------------------------------- | ---------------------------- |
| 核心波動       | 臺指選擇權波動率指數（台版 VIX） | 恐慌程度與波動 regime 主判斷 |
| 核心選擇權     | 臺指選擇權 PCR（成交量口徑）     | 交易情緒偏多或偏空           |
| 核心選擇權     | 臺指選擇權 PCR（OI 口徑）        | 持有偏好偏多或偏空           |
| 輔助資金流     | USD/TWD 匯率趨勢                 | 外資資金流向間接觀測         |
| 市場行為 proxy | 近 5 日實現波動率                | 短期波動擴張或收斂           |

#### US 市場

| 分類           | 指標                                   | 用途                                            |
| -------------- | -------------------------------------- | ----------------------------------------------- |
| 核心波動       | VIX                                    | 恐慌程度與波動 regime 主判斷                    |
| 輔助（有條件） | VVIX                                   | VIX 本身的波動率；僅在 VIX 快速拉升時觀察加速度 |
| 核心選擇權     | Put/Call Ratio（Volume / OI 雙口徑）   | 風險偏好判讀                                    |
| 核心選擇權     | ATM IV（`atm_iv_near`, `atm_iv_next`） | 近期隱含波動率水準                              |
| 核心選擇權     | IV Term Structure（`iv_term_slope`）   | 期限結構斜率，觀察近遠期 IV 差異                |
| 核心選擇權     | IV Skew（`iv_rr_25d`, `iv_bf_25d`）    | 尾部風險偏好（RR = 方向偏好，BF = 尾部厚薄）    |
| 市場行為 proxy | 近 5 日實現波動率                      | 短期波動擴張或收斂                              |

### 4.3 輸出結構

Psychology Analyst 報告應包含以下結構化段落：

1. **當前心理狀態**（頂層，4 狀態）：恐慌 / 樂觀 / 觀望 / 轉折
   - 綜合所有指標後的整體市場心理判斷
2. **波動 regime**：擴張 / 收斂 / 正常
3. **選擇權情緒**（options 子維度）：偏避險 / 中性 / 偏投機
   - 需多項訊號同向才可判斷；單一指標不可單獨決定（Tier C 禁止）
4. **市場行為 proxy 摘要**（`market_behavior_proxy`）：近 5 日實現波動率，說明是否出現異常訊號
5. **與技術面是否一致**：心理狀態與價格趨勢是否背離
6. **主要風險提示**：若有異常、訊號衝突或資料缺失，特別說明

## 五、指標判讀建議

### 5.1 台版 VIX 判讀

判讀台版 VIX 時，**不應使用固定數字門檻**。原因：

- 台版 VIX 歷史分布與 CBOE VIX 不同，直接套用 CBOE 門檻（如 < 15 / > 30）會失真
- 不同市場結構期間（低波動年度 vs 高波動年度）的基準線不同
- 固定門檻無法反映當前市場的相對極端程度

**建議判讀方式（percentile / z-score 導向）**：

| 判讀維度     | 方法                                    | 誩明                                        |
| ------------ | --------------------------------------- | ------------------------------------------- |
| 波動是否偏高 | 歷史 percentile（建議 1 年或 5 年視窗） | percentile > 75 → 偏高；< 25 → 偏低         |
| 偏離程度     | z-score（相對歷史均值）                 | z-score > 1.5 → 顯著偏高；< −1.5 → 顯著偏低 |
| 快速拉升     | 5 日或 10 日變化率                      | 短期急升需特別標注風險                      |

> **US VIX 同理**：固定門檻（如 VIX > 30 = 恐慌）僅能作為口語參考，不可用於自動化判讀規則，應以歷史分位數校準後取代。

### 5.2 Put/Call Ratio 判讀

| PCR 方向       | 判讀                         |
| -------------- | ---------------------------- |
| PCR 顯著上升   | Put 交易增加，市場偏避險     |
| PCR 顯著下降   | Call 交易增加，市場偏投機    |
| PCR 於極值反轉 | 可能為情緒轉折點（逆向觀察） |

- 建議同時觀察成交量口徑與 OI 口徑，若方向一致可信度較高
- 不應單獨使用 PCR 判斷方向，需與波動指標交叉驗證
- **結構性 flow 警示**：PCR 可能因結構性交易行為失真，例如：
  - 大量 covered call 策略 → 壓低 PCR，但不代表偷多
  - 法人避險需求集中 → 拉高 PCR，但不代表散戶恐慌
  - 發生上述情形時，應以 IV 指標作為主要判讀，PCR 降格為輔助參考

### 5.3 市場行為 proxy 判讀

| proxy 指標        | 異常條件               | 可能含義             |
| ----------------- | ---------------------- | -------------------- |
| 近 5 日實現波動率 | 顯著高於 20 日均值     | 短期情緒劇烈波動     |

> 上述門檻為初步參考，需根據歷史資料回測調整。

## 六、分析流程設計

### Step 1：取得資料

- TW：抓取台版 VIX 日線與歷史視窗（建議至少 1 年）、PCR（OpenAPI）、加權指數近 20 日 OHLCV、USD/TWD 匯率
- US：抓取 VIX、VVIX、個股或指數 options 資料
- 若核心 VIX（TW: TAIFEX VIX / US: ^VIX）缺失：
  - 標記 `degraded_mode=true`
  - 在 `warnings` 欄位具體揭露缺失欄位名稱與影響範圍
  - 後續各步驟輸出須使用 `insufficient evidence` 語氣，禁止在資料缺失時硬下方向性結論
  - 降級至 proxy-only 模式（僅使用市場行為 proxy 輔助判讀）

### Step 2：計算衍生指標

- 計算加權指數近 5 日實現波動率

### Step 3：規則判讀

以 deterministic rule 先產出訊號，不直接依賴 LLM 判斷。

**建議規則範例**：

- `VIX percentile > 75`（相對歷史高位）且 `PCR(OI) 高於歷史中位` → 偏恐慌
- `VIX percentile < 25`（相對歷史低位）且 PCR 雙口徑同步偏低 → 偏樂觀，但需警惕自滿
- `實現波動率 > 20 日均值 × 1.5` → 短期波動擴張
- `USD/TWD 連續 5 日升值` → 外資可能淨流出，risk-off 觀察

### Step 4：產生結構化摘要

依照 4.3 輸出結構，生成各段落判讀結果。

### Step 5：生成 `psychology_report`

可由 LLM 將規則化結果整理成自然語言，但不可改寫核心數值與訊號方向。

## 七、報告格式規範

```markdown
### 市場心理分析師報告

**分析日期**：2026-04-15
**標的**：2330（TW Market）

#### 摘要

- 整體判讀：核心指標偏向 [恐慌/樂觀/觀望/轉折]，輔助指標 [支持/部分背離]，目前可信度為 [高/中/低]
- 台版 VIX 目前位於歷史 [percentile]，屬於 [高位/中性/低位] 波動區間
- PCR（量 / OI）方向 [一致/不一致]，選擇權情緒偏 [避險/中性/投機]

#### 波動 Regime

- 台版 VIX 目前為 XX，近 20 日均值為 YY，波動狀態為 [擴張/收斂/正常]

#### 選擇權情緒

- PCR（成交量口徑）：X.XX → 判讀為 [偏避險/偏投機/中性]
- PCR（OI 口徑）：X.XX → 判讀為 [偏避險/偏投機/中性]
- 兩種口徑方向 [一致/不一致]

#### 資金流 Proxy

- USD/TWD：近 5 日趨勢為 [升值/貶值/持平]，暗示 [risk-on/risk-off/中性]

#### 市場行為 Proxy

- 近 5 日實現波動率：X.XX%（20 日均值 Y.YY%）

#### 與技術面一致性

- [一致 / 背離]，說明理由

#### 風險提示

- ...
```

### 報告要求

- 摘要優先寫出 3 個最重要訊號
- 摘要必須先做整合判讀（回答「總體市場心理狀態是什麼」），不可只逐條列指標
- 每個結論盡量附對應指標依據
- 避免使用模糊措辭，例如「感覺偏多」
- 若資料不足，必須明確標示可信度限制
- 市場行為 proxy 段落需說明其為間接觀測，非直接情緒量測

## 八、與其他分析師邊界

| 類別                                    | 分析師             |
| --------------------------------------- | ------------------ |
| 價格趨勢 / 動能 / 量價                  | Technical Analyst  |
| 波動 / 恐慌 / 風險偏好 / 市場行為 proxy | Psychology Analyst |
| 三大法人 / 融資融券 / 借券 / 權證       | Chip Analyst       |
| 新聞事件解讀 / 新聞語氣 / 社群情緒      | News Analyst       |

### 邊界說明

- **新聞語氣**歸 News Analyst，Psychology Analyst 不使用任何文本情緒分析
- **VIX 與 PCR** 雖也出現在 Chip Analyst 的大盤籌碼（全市場 PCR 作基準線），但用途不同：
  - Chip Analyst 用 PCR 作為大額交易人 PCR 的對照基準
  - Psychology Analyst 用 PCR 作為整體市場情緒判讀
- **匯率**不算籌碼核心，放在 Psychology Analyst 作為輔助資金流 proxy
- **市場行為 proxy**（實現波動率）為 Psychology Analyst 輔助維度

## 九、推薦落地方案

1. TW 市場優先用台版 VIX + 臺指選擇權 PCR 作為核心判讀
2. 心理狀態採加權整合：核心指標（VIX、PCR）權重 2，輔助指標（USD/TWD、實現波動率）權重 1
3. USD/TWD 匯率作為資金流間接觀測
4. US 市場以 VIX、Put/Call Ratio、ATM IV（`atm_iv_near`/`atm_iv_next`）、IV Term Structure（`iv_term_slope`）、IV Skew（`iv_rr_25d`/`iv_bf_25d`）為主；VVIX 僅在 VIX 快速拉升時有條件使用
5. 嚴禁單一指標獨立決策（Tier C），至少需 2 個以上同向訊號才輸出方向性結論
6. 以 regime 判讀為核心，不與籌碼資料混用
7. 不使用新聞語氣或社群情緒，與 News Analyst 職責分開
8. 產出 `psychology_report`，供後續研究員與交易員交叉閱讀

## 十、觀念校正

以下為常見誤解，實作與判讀時應主動避免：

- **Volatility ≠ Sentiment**：波動率（特別是 IV）反映的是「風險定價 + 不確定性」，不等於純情緒。IV 高不代表市場一定恐慌，也可能是已知事件的風險溢酬（如財報、選舉）。
- **單一指標不可獨立下結論**：PCR 或 IV 都可能受結構性 flow 影響，需交叉驗證，禁止以任一指標單獨給出方向性判斷（Tier C 禁止）。
- **固定門檻不可跨市場直接套用**：不同市場波動結構不同，建議以歷史分布校準（percentile/z-score），台版 VIX 不可直接套用 CBOE VIX 門檻。
- **不依賴新聞語氣或社群情緒**：Psychology Analyst 只使用量化市場指標，新聞情緒由 News Analyst 處理。下游 Masters / Researchers 會整合兩份報告，Psychology Analyst 不需要、也不應重複此功能。

## 十一、指標證據等級

### Tier A（可直接採用，理論與實務一致）

- US：`atm_iv_near`, `atm_iv_next`, `iv_term_slope`, `iv_rr_25d`, `iv_bf_25d`
- US：`put_call_ratio_volume`, `put_call_ratio_oi`
- TW：`taifex_vix`, `pcr_volume`, `pcr_oi`

### Tier C（禁止直接決策）

- 單獨用某一指標（例如只看 PCR、只看 MA 偏離、只看 VIX 絕對值）直接給出方向性結論
- 使用未經歷史校準的固定數字門檻作為判讀規則
