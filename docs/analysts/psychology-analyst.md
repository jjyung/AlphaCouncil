# 市場心理分析師 (Psychology Analyst) 開發方式

## 文件目的

本文件定義 AlphaCouncil 中市場心理分析師的角色定位、資料策略、指標設計與輸出格式。

市場心理分析師關注的是**市場在害怕什麼、偏好什麼、風險承受度如何變化**，以「可量化的數字型指標」為核心判讀工具，不處理法人流向、融資融券或借券賣出等籌碼資訊，也不處理新聞語氣或社群情緒（該部分由 News Analyst 負責）。

---

## 一、角色定位

市場心理分析師負責從波動指標、選擇權情緒、市場行為 proxy 與資金流向 proxy，判斷目前市場屬於恐慌、樂觀、觀望或風險轉折階段。

### 主要職責

- 分析市場波動是否擴張或收斂
- 分析風險偏好是 risk-on 還是 risk-off
- 分析選擇權市場反映的多空情緒
- 監控市場行為 proxy 指標的異常變化
- 產出 `psychology_report`

### 不負責事項

- 不分析誰在買、誰在賣（歸 Chip Analyst）
- 不處理三大法人、融資融券、借券賣出（歸 Chip Analyst）
- 不取代技術分析師的價格趨勢判斷（歸 Technical Analyst）
- 不解讀新聞語氣或社群情緒（歸 News Analyst）

## 三、資料來源策略

### 3.1 US 市場

- `yfinance`：取得 VIX (`^VIX`)、VVIX (`^VVIX`)、個股與 ETF options 資料
- VIX / VVIX：作為恐慌與波動 regime 主指標
- Put/Call Ratio：從 options chain 計算，作為風險偏好輔助訊號
- Options Skew：觀察 put 與 call 之間的隱含波動率差異

### 3.2 TW 市場

台股有免費的官方心理指標可用，以 TAIFEX 提供的波動率指數與選擇權情緒資料為核心，搭配市場行為 proxy 補足。

#### 核心波動指標：TAIFEX 臺指選擇權波動率指數（台版 VIX）

- TAIFEX 提供「臺指選擇權波動率指數」，以 CBOE VIX 公式計算
- **即時/分鐘資料**：`https://www.taifex.com.tw/cht/7/vixMinNew`
- **每日歷史資料**：`https://www.taifex.com.tw/cht/7/vixDaily3MNew`
- 此為台股 Psychology Analyst 的**首要核心指標**
- 免費、官方、無需 API 金鑰

#### 核心選擇權情緒：TAIFEX Put/Call Ratio

- **端點**：`https://www.taifex.com.tw/cht/3/pcRatio`
- **下載頁**：`https://www.taifex.com.tw/cht/3/pcRatioDown`
- 提供臺指選擇權 Put/Call 比，含兩種口徑：
  - 成交量口徑：反映交易情緒
  - 未平倉量（OI）口徑：反映持有偏好
- 免費、官方、無需 API 金鑰

#### 輔助資金流 proxy：USD/TWD 匯率趨勢

- `yfinance`：`USDTWD=X` 取得匯率日線
- 匯率不算籌碼核心，定位為資金流向的間接觀測
- USD/TWD 持續升值（台幣貶值）可能暗示外資撤出、risk-off
- USD/TWD 持續貶值（台幣升值）可能暗示外資流入、risk-on

#### 市場行為 proxy（數字型）

以加權指數（`^TWII`）為基礎計算，從市場行為側面反映情緒狀態：

- **加權指數近 5 日實現波動率**：衡量短期波動是否擴張
- **MA5 乖離率**：`(close - MA5) / MA5`，衡量短線偏離程度
- **上漲/下跌家數比**：來自 TWSE 每日市場統計
- **漲停/跌停家數**：來自 TWSE 每日市場統計
- **跳空缺口頻率**：近 5 日加權指數出現跳空缺口的次數
- **近 5 日正負報酬交替次數**：衡量市場猶豫程度

這些指標以 5 日（一週）為觀察視窗，定位為 **secondary proxy**，輔助核心波動與選擇權指標。

### 3.3 資料來源端點總整理

| 指標 | 分類 | 資料來源 | 端點 / 方法 | 頻率 | 需要 API Key |
|------|------|----------|-------------|------|-------------|
| 臺指選擇權波動率指數 | 核心 | TAIFEX | `/cht/7/vixDaily3MNew` | 日 | 否 |
| 臺指選擇權 PCR（成交量） | 核心 | TAIFEX | `/cht/3/pcRatioDown` | 日 | 否 |
| 臺指選擇權 PCR（OI） | 核心 | TAIFEX | `/cht/3/pcRatioDown` | 日 | 否 |
| USD/TWD 匯率 | 輔助 | yfinance | `yf.Ticker("USDTWD=X")` | 日 | 否 |
| 加權指數 OHLCV | 計算用 | yfinance | `yf.Ticker("^TWII")` | 日 | 否 |
| 上漲/下跌家數 | 輔助 | TWSE | `/exchangeReport/MI_INDEX` | 日 | 否 |
| 漲停/跌停家數 | 輔助 | TWSE | `/exchangeReport/MI_INDEX` | 日 | 否 |
| VIX (US) | 核心 | yfinance | `yf.Ticker("^VIX")` | 日 | 否 |
| VVIX (US) | 輔助 | yfinance | `yf.Ticker("^VVIX")` | 日 | 否 |

### 3.4 選型原則

- 心理分析以**數字型指標**為核心，不依賴新聞語氣或社群情緒
- 台股已有官方免費 VIX 指標，不再需要純 proxy 替代
- 心理分析重點是 regime 判讀，不追求單一完美指標
- 台股 secondary proxy 需在報告中說明其為間接觀測
- 心理分析只回答風險偏好，不回答籌碼流向

## 四、分析框架

### 4.1 核心問題

- 市場目前偏恐慌還是偏樂觀？
- 波動是否明顯放大？
- 選擇權市場偏避險還是偏投機？
- 市場行為 proxy 是否出現異常？
- 短期情緒是否與價格趨勢背離？

### 4.2 指標分組

#### TW 市場

| 分類 | 指標 | 用途 |
|------|------|------|
| 核心波動 | 臺指選擇權波動率指數（台版 VIX） | 恐慌程度與波動 regime 主判斷 |
| 核心選擇權 | 臺指選擇權 PCR（成交量口徑） | 交易情緒偏多或偏空 |
| 核心選擇權 | 臺指選擇權 PCR（OI 口徑） | 持有偏好偏多或偏空 |
| 輔助資金流 | USD/TWD 匯率趨勢 | 外資資金流向間接觀測 |
| 市場行為 proxy | 近 5 日實現波動率 | 短期波動擴張或收斂 |
| 市場行為 proxy | MA5 乖離率 | 短線偏離程度 |
| 市場行為 proxy | 上漲/下跌家數比 | 市場廣度 |
| 市場行為 proxy | 漲停/跌停家數 | 極端情緒信號 |
| 市場行為 proxy | 跳空缺口頻率 | 短期不連續性 |
| 市場行為 proxy | 近 5 日正負報酬交替次數 | 市場猶豫程度 |

#### US 市場

| 分類 | 指標 | 用途 |
|------|------|------|
| 核心波動 | VIX | 恐慌程度與波動 regime 主判斷 |
| 核心波動 | VVIX | VIX 的波動率，觀察二階不確定性 |
| 核心選擇權 | Put/Call Ratio | 風險偏好判讀 |
| 核心選擇權 | Options Skew | 尾部風險偏好 |
| 市場行為 proxy | 近 5 日實現波動率 | 短期波動擴張或收斂 |

### 4.3 輸出結構

Psychology Analyst 報告應包含以下結構化段落：

1. **當前心理狀態**：恐慌 / 樂觀 / 觀望 / 轉折
2. **波動 regime**：擴張 / 收斂 / 正常
3. **選擇權情緒**：偏避險 / 偏投機 / 中性
4. **市場行為 proxy 摘要**：是否出現異常訊號
5. **與技術面是否一致**：心理狀態與價格趨勢是否背離
6. **主要風險提示**：若有異常或風險因子，特別說明

## 五、指標判讀建議

### 5.1 台版 VIX 判讀

| VIX 區間 | 判讀 |
|----------|------|
| < 15 | 低波動，市場偏樂觀或自滿 |
| 15–25 | 正常區間 |
| 25–35 | 波動升溫，市場偏謹慎 |
| > 35 | 高波動，市場偏恐慌 |

> 上述門檻為參考值，實際門檻應根據台版 VIX 歷史分布調整。

### 5.2 Put/Call Ratio 判讀

| PCR 方向 | 判讀 |
|----------|------|
| PCR 顯著上升 | Put 交易增加，市場偏避險 |
| PCR 顯著下降 | Call 交易增加，市場偏投機 |
| PCR 於極值反轉 | 可能為情緒轉折點（逆向觀察） |

- 建議同時觀察成交量口徑與 OI 口徑，若方向一致可信度較高
- 不應單獨使用 PCR 判斷方向，需與波動指標交叉驗證

### 5.3 市場行為 proxy 判讀

| proxy 指標 | 異常條件 | 可能含義 |
|------------|----------|----------|
| 近 5 日實現波動率 | 顯著高於 20 日均值 | 短期情緒劇烈波動 |
| MA5 乖離率 | 偏離超過 ±3% | 短線過熱或過冷 |
| 上漲/下跌家數比 | < 0.5 或 > 2.0 | 極端一面倒 |
| 漲停/跌停家數 | 同日漲停或跌停 > 30 家 | 極端情緒信號 |
| 跳空缺口頻率 | 5 日內出現 ≥ 3 次缺口 | 市場不連續性升高 |
| 正負報酬交替次數 | 5 日內交替 ≥ 4 次 | 市場猶豫、無明確方向 |

> 上述門檻為初步參考，需根據歷史資料回測調整。

## 六、分析流程設計

### Step 1：取得資料

- TW：抓取台版 VIX 日線、PCR、加權指數近 20 日 OHLCV、上漲下跌家數、USD/TWD 匯率
- US：抓取 VIX、VVIX、個股或指數 options 資料
- 若 TAIFEX VIX 資料暫時無法取得，標記為資料缺失，降級至 proxy-only 模式

### Step 2：計算衍生指標

- 計算加權指數近 5 日實現波動率
- 計算 MA5 乖離率
- 計算跳空缺口頻率
- 計算近 5 日正負報酬交替次數

### Step 3：規則判讀

以 deterministic rule 先產出訊號，不直接依賴 LLM 判斷。

**建議規則範例**：

- `VIX > 30` 且 `PCR(OI) > 1.2` → 偏恐慌
- `VIX < 15` 且漲停家數多 → 偏樂觀，但需警惕自滿
- `實現波動率 > 20 日均值 × 1.5` → 短期波動擴張
- `MA5 乖離 > 3%` 且 `PCR 下降` → 短線過熱
- `正負報酬交替 ≥ 4` → 市場觀望
- `USD/TWD 連續 5 日升值` → 外資可能淨流出，risk-off 觀察

### Step 4：產生結構化摘要

依照 4.3 輸出結構，生成各段落判讀結果。

### Step 5：生成 `psychology_report`

可由 LLM 將規則化結果整理成自然語言，但不可改寫核心數值與訊號方向。

## 七、報告格式規範

```markdown
### 市場心理分析師報告

**分析日期**：2026-04-15
**標的**：2330（TW Market）

#### 摘要
- 台版 VIX 處於正常偏低區間，市場偏樂觀
- 選擇權 PCR 成交量口徑略降，交易偏投機
- 近 5 日上漲家數比持續偏高，但無極端訊號

#### 波動 Regime
- 台版 VIX 目前為 XX，近 20 日均值為 YY，波動狀態為 [擴張/收斂/正常]

#### 選擇權情緒
- PCR（成交量口徑）：X.XX → 判讀為 [偏避險/偏投機/中性]
- PCR（OI 口徑）：X.XX → 判讀為 [偏避險/偏投機/中性]
- 兩種口徑方向 [一致/不一致]

#### 資金流 Proxy
- USD/TWD：近 5 日趨勢為 [升值/貶值/持平]，暗示 [risk-on/risk-off/中性]

#### 市場行為 Proxy
- 近 5 日實現波動率：X.XX%（20 日均值 Y.YY%）
- MA5 乖離率：X.XX%
- 上漲/下跌家數比：X.XX
- 漲停/跌停家數：X / Y
- 跳空缺口：近 5 日 N 次
- 正負報酬交替：近 5 日 N 次

#### 與技術面一致性
- [一致 / 背離]，說明理由

#### 風險提示
- ...
```

### 報告要求

- 摘要優先寫出 3 個最重要訊號
- 每個結論盡量附對應指標依據
- 避免使用模糊措辭，例如「感覺偏多」
- 若資料不足，必須明確標示可信度限制
- 市場行為 proxy 段落需說明其為間接觀測，非直接情緒量測

## 八、與其他分析師邊界

| 類別 | 分析師 |
|------|--------|
| 價格趨勢 / 動能 / 量價 | Technical Analyst |
| 波動 / 恐慌 / 風險偏好 / 市場行為 proxy | Psychology Analyst |
| 三大法人 / 融資融券 / 借券 / 權證 | Chip Analyst |
| 新聞事件解讀 / 新聞語氣 / 社群情緒 | News Analyst |

### 邊界說明

- **新聞語氣**歸 News Analyst，Psychology Analyst 不使用任何文本情緒分析
- **VIX 與 PCR** 雖也出現在 Chip Analyst 的大盤籌碼（全市場 PCR 作基準線），但用途不同：
  - Chip Analyst 用 PCR 作為大額交易人 PCR 的對照基準
  - Psychology Analyst 用 PCR 作為整體市場情緒判讀
- **匯率**不算籌碼核心，放在 Psychology Analyst 作為輔助資金流 proxy
- **市場行為 proxy**（實現波動率、乖離率、漲跌家數等）為 Psychology Analyst 獨有

---

## 九、推薦落地方案

1. TW 市場優先用台版 VIX + 臺指選擇權 PCR 作為核心判讀
2. 市場行為 proxy（實現波動率、乖離率、漲跌家數等）作為輔助佐證
3. USD/TWD 匯率作為資金流間接觀測
4. US 市場以 VIX、VVIX、Put/Call Ratio、Options Skew 為主
5. 以 regime 判讀為核心，不與籌碼資料混用
6. 不使用新聞語氣或社群情緒，與 News Analyst 職責分開
7. 產出 `psychology_report`，供後續研究員與交易員交叉閱讀
