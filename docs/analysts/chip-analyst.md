# 籌碼分析師 (Chip Analyst) 開發方式

## 文件目的

本文件定義 AlphaCouncil 中籌碼分析師的角色定位、資料來源、分析範圍與輸出方式。

籌碼分析師關注的是**誰在買、誰在賣、資金流往哪裡去、持有結構如何變化**，與市場心理分析師的波動/情緒判讀分開處理。

本角色內部再拆成兩個分析域：

- **大盤籌碼**：觀察指數與全市場方向，用於判斷全局資金風向
- **個股籌碼**：觀察單一股票的法人現貨與權證結構，用於判斷個股支撐與短線賣壓

---

## 一、角色定位

籌碼分析師負責從法人流向、衍生品部位、現貨買賣超、權證活動、融資融券、借券賣出與持股結構，判讀目前市場參與者的行為是否支持或壓抑價格趨勢。

### 主要職責

- 分析大盤法人與衍生品主要部位
- 分析個股法人現貨買賣超與連買連賣
- 分析權證是否形成短線隔日沖賣壓
- 分析融資融券、借券與持股結構變化
- 產出 `chip_report`

### 不負責事項

- 不處理 MA、MACD、RSI、KD 等技術指標
- 不處理 VIX、恐慌指數、風險偏好等心理指標
- 不解讀基本面與新聞事件本身

---

## 二、輸入與輸出

| 欄位 | 型別 | 說明 |
|------|------|------|
| `ticker` | `str` | 股票代碼 |
| `date` | `str` | 分析日期 |
| `market` | `"us" \| "tw"` | 市場別 |

| 輸出欄位 | 型別 | 說明 |
|----------|------|------|
| `chip_report` | `str` | 籌碼分析報告 |

## 三、資料來源策略

### 3.1 TW 市場

台股籌碼資料支援度高，建議以官方來源為主。

#### 大盤籌碼

- TAIFEX：三大法人期貨 / 選擇權部位
- TAIFEX：外資期貨未平倉
- TAIFEX：大額交易人選擇權未沖銷部位
- TAIFEX：全市場 Put/Call Ratio（作為基準線）

#### 個股籌碼

- TWSE OpenAPI：融資融券、三大法人
- TPEX OpenAPI：上櫃融資融券、三大法人、借券賣出
- TWSE / TPEX 權證相關頁面或下載檔：觀察權證活躍度與短線賣壓 proxy

### 3.2 US 市場

美股免費籌碼資料相對分散，建議以輔助性資料為主：

- `yfinance`：short interest、institutional holders、major holders
- ETF / fund flow 資料（若有免費來源）
- 特定標的可補 institutional ownership 變化

### 3.3 選型原則

- 台股優先使用官方資料，因籌碼欄位完整且具市場特色
- 美股以 short interest 與機構持股作為輔助，不追求完全對齊台股口徑
- 籌碼分析重點在參與者結構與資金方向，不等同市場心理
- 指數衍生品資料只用於**大盤籌碼**，不可直接套用到個股
- 個股分析不使用分點資訊，避免資料成本過高與品質不穩
- 個股權證資料定位為**短線壓力輔助訊號**，不作為主方向訊號

## 四、分析框架

### 4.1 大盤籌碼

#### 核心問題

- 大盤目前是偏多還是偏空？
- 外資在指數期貨的主方向是否明確？
- 法人期貨部位是否支持現貨趨勢？
- 大額交易人選擇權部位是否顯示避險或偏多押注？

#### 核心指標

| 指標類別 | 指標 | 說明 |
|----------|------|------|
| 法人期貨 | 三大法人期貨多空淨額 | 觀察法人整體方向 |
| 外資期貨 | 外資未平倉多方 / 空方 / 淨額 | 觀察外資主部位 |
| 大戶選擇權 | 大額交易人未沖銷部位 Put/Call Ratio | 觀察大戶避險或偏多偏空傾向 |
| 市場基準 | 全市場 Put/Call OI Ratio | 對照大戶口徑，避免誤判 |

#### 指標口徑

- `三大法人買賣超` 在期貨情境下建議改稱 **多空淨額**
- `外資期貨未平倉` 必須區分：多方、空方、淨額
- `大額交易人 Put/Call Ratio` 建議由未沖銷部位自行計算，不直接等同官方市場 PCR

#### 建議公式

- `foreign_futures_oi_net = foreign_long_oi - foreign_short_oi`
- `large_trader_pcr_top5 = top5_put_oi / top5_call_oi`
- `market_pcr_oi = market_put_oi / market_call_oi`

### 4.2 個股籌碼

#### 核心問題

- 趨勢背後是否有法人資金支持？
- 哪一類法人正在買進或賣出？
- 法人買超是否具有連續性？
- 權證活動是否可能帶來隔日沖賣壓？

#### 核心指標

| 指標類別 | 指標 | 說明 |
|----------|------|------|
| 現貨法人 | 外資 / 投信 / 自營商買賣超 | 個股主訊號 |
| 趨勢延續 | 外資 / 投信 / 自營商連買連賣 | 觀察籌碼是否延續 |
| 槓桿 / 放空 | 融資融券、借券賣出 | 輔助判斷追價與壓力 |
| 權證結構 | 權證買超、成交活躍度、偏多權證占比 | 短線風險提示 |

#### 權證定位

- 權證資料不用來判斷中期主方向
- 若權證買入明顯升高，常代表短線交易資金活躍
- 權證偏多且短線漲幅已大時，需提示**隔日沖與獲利了結賣壓**風險
- 若法人現貨買超與權證活躍方向一致，可信度較高
- 若權證偏多但法人現貨不跟，應視為短線雜訊或過熱警訊

### 4.3 指標整理

| 分析域 | US 市場 | TW 市場 |
|--------|---------|---------|
| 大盤籌碼 | Institutional / index derivatives（若可得） | 三大法人期貨、外資期貨未平倉、大額交易人選擇權 OI |
| 個股籌碼 | Institutional holders、short interest | 三大法人現貨買賣超、融資融券、借券、權證 |

### 4.4 TW 特有判讀

- 外資期貨未平倉淨額偏多，較適合解讀為**大盤全局偏多**，不是單一股票訊號
- 大額交易人選擇權 PCR 偏低或偏高，較適合解讀為**大戶部位傾向與避險結構**
- 外資連買連賣個股可作為趨勢延續或反轉觀察點
- 融資快速升高常代表散戶追價與槓桿升溫
- 借券賣出增長可能代表中短期空方壓力上升
- 權證偏多活躍可視為短線資金聚焦，但需同時提示隔日沖賣壓風險

## 五、與其他分析師邊界

| 類別 | 分析師 |
|------|--------|
| OHLCV / 技術指標 | Technical Analyst |
| 恐慌 / 波動 / 風險偏好 | Psychology Analyst |
| 法人 / 融資融券 / 借券 / 權證 | Chip Analyst |
| 新聞事件 | News Analyst |

---

## 六、推薦落地方案

1. 先把 `chip_report` 內部分成「大盤籌碼」與「個股籌碼」兩節
2. 大盤籌碼以 TAIFEX 期貨 / 選擇權官方資料為主
3. 個股籌碼以 TWSE / TPEX 三大法人現貨買賣超為主
4. 個股不納入分點資訊，降低資料蒐集成本與維護難度
5. 權證資料只作為短線賣壓與交易過熱風險提示
